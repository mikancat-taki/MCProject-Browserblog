🧭 コンセプト

MCBrowser はシンプルさと美しさを追求した Windows 向けの軽量ブラウザです。 余計な機能を省き、あなたの検索・閲覧体験をスムーズにします。

💡 主な特徴

  • ⚡ 素早い起動と軽快な動作
  • 🎨 テーマ・背景のカスタマイズ対応
  • 🕒 シンプルなホーム画面と時計機能
  • 🔍 Google 検索に完全対応

📸 スクリーンショット

🧱 技術構成

MCBrowser は Electron と HTML / CSS / JavaScript で構築されています。 デスクトップ環境でネイティブのように動作し、設定はローカル保存されます。
